# Supabase Template
This is a NextJS raw template, Other templates are derived from this template.

## Tech Stack
- Next.js
- React
- TypeScript
- Tailwind CSS
- React Icons
- Supabase

## Configurations
- Biome
- React Compiler
- View Transitions

## Environment 

> [!TIP]
> Needed: `Supabase Keys`

`.env.local`
```
NEXT_PUBLIC_SUPABASE_URL=""
NEXT_PUBLIC_SUPABASE_ANON_KEY=""
```
