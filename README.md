# Supabase Template
A NextJS Template with Supabase helper functions, inherited from [Raw Template](https://github.com/templatable/raw)

## Tech Stack
- Next.js
- React
- TypeScript
- Tailwind CSS
- React Icons
- Supabase

## Configurations
- Biome
- React Compiler
- View Transitions

## Environment 

> [!TIP]
> Needed: `Supabase Keys`

`.env.local`
```
NEXT_PUBLIC_SUPABASE_URL=""
NEXT_PUBLIC_SUPABASE_ANON_KEY=""
```
